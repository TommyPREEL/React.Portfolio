## ⚡ Quick Start (5 Minutes)

### Step 1: Customize Content
```bash
# Edit this file with YOUR information
src/i18n.ts

# Change:
# - Your name & description in 'about'
# - Your projects in 'projects.items'
# - Your skills in 'skills.categories'
# - Your contact info in 'contact.methods'
```

### Step 2: Run Locally
```bash
npm run dev
# Opens at http://localhost:5173
```

### Step 3: Test
- Use WASD/ZQSD to move the car
- Click zones to view content
- Click language button to switch EN/FR
- Test all links work

### Step 4: Build & Deploy
```bash
npm run build
# Upload dist/ to Vercel, Netlify, or GitHub Pages
```

## 🎨 Customization Options

### Content (5 minutes)
- Edit `src/i18n.ts`
- Replace sample content with your info
- That's it!

### Colors (2 minutes)
- Edit `src/App.css` - `:root` section
- Change CSS variables
- Colors update instantly

### 3D Layout (5 minutes)
- Edit `src/App.tsx` - Zone components
- Adjust positions: `position={[x, y, z]}`
- Change zone sizes/colors

### Add Sections (15 minutes)
- Add to Content interface in `i18n.ts`
- Create render function in `ContentPanel.tsx`
- Add Zone in `App.tsx` World component
- Add CSS styling in `App.css`
